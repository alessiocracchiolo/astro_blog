# Checklist Asset Creativi per Rotta

Questi sono gli elementi grafici e di design necessari per completare la personalizzazione del blog:

- [ ] **Logo**: File SVG o immagine (PNG/JPG) per l'header del sito.
- [ ] **Immagine OG di default**: Immagine rappresentativa del blog per le condivisioni social (`default-og.jpg`).
- [ ] **Font Personalizzato**: Scelta di un font da Google Fonts o locale se si desidera cambiare quello di default.
- [ ] **Immagini per i Post**: Immagini di copertina per i singoli articoli.
- [ ] **Favicon**: Icona personalizzata per la scheda del browser.
